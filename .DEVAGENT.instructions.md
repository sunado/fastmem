  This agent provides comprehensive development environment management for the FastMem flashcard application (SvelteKit + Vite + TypeScript + Drizzle ORM + SQLite).
  
  ## Core Capabilities
  
  ### 1. Development Server Management
  - Start: Run `npm run dev` (Vite dev server on http://localhost:5173)
  - Stop: Kill vite processes
  - Status: Check if running and respond with URL
  - Restart: Stop then start
  
  ### 2. Database Operations
  - Initialize: `npm run db:init` (creates SQLite DB + seed data)
  - Query: Execute SQL via `sqlite3 data/fastmem.db`
  - Status: Report user count, table counts
  - Reset: Delete DB file (destructive, confirm first)
  
  ### 3. Build & Quality Checks
  - Production build: `npm run build`
  - Type checking: `npm run check` (one-time) or `npm run check:watch` (continuous)
  - Linting: `npm run lint`
  - Formatting: `npm run format`
  - Preview: `npm run preview`
  
  ### 4. Project Initialization
  - First-time setup: Install npm dependencies + initialize database
  - Verify prerequisites: Node.js v18+, npm v8+, Git

# Dev Environment Tools for FastMem

Complete agent tools for managing the FastMem development environment. Use these tools to set up, manage, and troubleshoot the development environment.

## Quick Start

```bash
# First time only:
npm install
npm run db:init
npm run dev

# Then open: http://localhost:5173
# Login: user / user
```

## Using Helper Scripts

### Interactive Helper (`scripts/dev.sh`)
```bash
./scripts/dev.sh help          # Show all commands
./scripts/dev.sh status        # Environment status
./scripts/dev.sh dev           # Start dev server
./scripts/dev.sh db:inspect    # Open SQLite CLI
./scripts/dev.sh db:query "SELECT COUNT(*) FROM users"
./scripts/dev.sh format        # Format code
./scripts/dev.sh lint          # Check code quality
```

### Agent Commands (`scripts/agent-commands.sh`)
```bash
./scripts/agent-commands.sh dev:start
./scripts/agent-commands.sh db:status
./scripts/agent-commands.sh build:prod
```

## VS Code Integration

Use the Task Panel (`Tasks: Run Task`) to access 14 development tasks:
- 🚀 Dev Server: Start/Stop
- 🗄️ Database: Initialize/Inspect/Query  
- 🏗️ Build: Production
- ✅ Check: Types (one-time and watch)
- 🔍 Lint: ESLint
- 💅 Format: Prettier
- 📊 Status monitoring

## Database Management

```bash
# Initialize with seed data
npm run db:init

# Inspect with SQLite CLI
sqlite3 data/fastmem.db

# Query directly
sqlite3 data/fastmem.db "SELECT * FROM users"

# Reset (deletes all data)
rm data/fastmem.db
npm run db:init
```
